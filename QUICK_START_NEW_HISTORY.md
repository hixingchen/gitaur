## 🐛 常见问题

### Q1: 界面显示不正常？
A1: 请确保重新构建项目：`npm run build`

### Q2: 搜索功能不工作？
A2: 请检查搜索关键词是否正确，支持模糊搜索

### Q3: 如何回退到旧界面？
A3: 在`App.tsx`中将导入改回原组件：
```typescript
const HistoryView = lazy(() => import('./components/Graph/HistoryView').then((m) => ({ default: m.HistoryView })));
```
